# ShiftWorker Constitution

_The guiding principles for building ShiftWorker — the free OpenClaw setup wizard._

## 1. Zero-CLI for End Users
The target user has never opened a terminal. Every interaction happens through a web UI. If they need to type a command, we failed.

## 2. Users Own Their Infrastructure
We never host user data or agents. We provision and configure on the user's own VPS account, then hand over the keys. Their server, their data, their control.

## 3. Provider Agnostic
Support multiple VPS providers starting with DigitalOcean as the primary provider. Additional providers (Hetzner, Vultr, Linode) may be added in the future. Never lock users into a single provider. Recommend the best value, but let them choose.

## 4. Free Core, Premium Extras
The basic setup wizard is always free. Revenue comes from premium skill packs, advanced integrations, support tiers, and provider referral commissions — never from gatekeeping the basics.

## 5. Transparent and Trustworthy
Users can see exactly what we're installing on their server. No hidden agents, no data collection without consent, no black boxes. Open source the provisioning scripts.

## 6. Community First
Build the largest OpenClaw onboarding community. Help people get started, share skills, and learn. The community IS the product — everything else follows.
